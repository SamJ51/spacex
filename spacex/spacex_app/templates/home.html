Welcome to the SpaceX Dashboard!

The Project's Purpose

The purpose of this project is to design and develop a Django-based web application that consumes and displays structured data from the SpaceX public API.

This data includes key information regarding:

  • Crew Members
  • Launches
  • Payloads

Main Features

Launches Page

The Launches page contains all SpaceX launches with each individual launch being represented as an individual card.

Crew Page

The Crew page contains all members of the SpaceX crew with each individual crew member being represented as a separate card.

Payloads Page

The Payloads page contains all of the payloads that have been sent into space by SpaceX. Each of these payloads is represented as an individual card.

Search

Search is available in the Launches, Payloads, and Crew pages. The search bar will filter cards based on whether the search text is present as a substring in the card.

Database Caching

All SpaceX data is retrieved from the SQLite database rather than from the API directly. This increases the website's refresh speed.

Expandable Card Sections

Cards from the Payloads and Launches pages have expandable card sections. This allows the user to have the choice of either keeping the card small, or expanding the card to get additional information.
